Computational Wave Flume status

I'll try putting a few things here to support discussion about where to focus with the numerical wavetank

Flume Geometries

  • Tank -- Rectangular with wave absorber at end
  • Beach -- Bakhtyar papers
  • Beach -- Beach Erosion Board
  • Levee -- Simple Tapezoidal
  • Levee -- Sacramento Pocket Area
  • Structure -- Corps Castle

    • Wave functions

    • Boundary and Initial Conditions
    • * 1D Monochromatic waves based on linear wave theory (general finite depth solution)
    • * 1D Nearly monochromatic wave groups based on linear wave theory
    • Initial Conditions Only
    • * JONSWAP spectrum
    • * Solitary wave (does not appear to be implemented correctly)
